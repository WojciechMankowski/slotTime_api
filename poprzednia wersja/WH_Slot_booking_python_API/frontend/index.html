Połączone z backendem FastAPI (API)
Krok 1 Logowanie

Zaloguj się jako admin (magazyn) lub client (klient).

Logowanie do systemu

Startowe konta: admin / admin oraz demo klient client1 / client1.

Role użytkowników

Dwa główne typy kont – magazyn i klienci.
  • Admin (magazyn) – tworzy sloty, zarządza szablonami, limitami dziennymi i klientami.
  • Client (klient) – rezerwuje sloty (limit dzienny Inbound/Outbound współdzielony), wysyła prośbę o anulację.
W tej wersji prototypu logowanie odbywa się bez SSO – bezpośrednio przez API.
